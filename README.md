# Верификация модели дисперсии в MEEP: отражение от границы воздух–кварц

Работу выполнили: Сафонов Тимофей, Попцов Тимофей, Кузьмин Дмитрий

## О проекте

В этом проекте выполняется верификация дисперсионной модели плавленого кварца в пакете **MEEP** на примере отражения электромагнитной волны от плоской границы раздела **воздух–кварц**.  
Численный спектр отражения, полученный методом **FDTD**, сравнивается с аналитическим решением по **формулам Френеля**. В основе постановки лежит учебный пример MEEP для **Reflectance Spectrum of Air-Silica Interface**: 1D-ячейка, источник с диапазоном **0.4–0.8 мкм**, два отдельных расчёта — для пустой ячейки и для ячейки с кварцем.

## Цель работы

Проверить, насколько корректно дисперсия плавленого кварца воспроизводится в MEEP, сравнив численный коэффициент отражения с аналитическим расчётом.

## Постановка задачи

В рамках проекта требуется:

- реализовать **1D-модель** границы воздух/плавленый кварц;
- задать дисперсию кварца;
- использовать **гауссов источник** с широким спектром в диапазоне **0.4–0.8 мкм**;
- выполнить два расчёта:
  - пустая ячейка для измерения падающего потока;
  - ячейка с кварцем для измерения отражённого потока;
- вычислить спектр отражения;
- сравнить численный результат с аналитической кривой по формулам Френеля;
- оценить относительную погрешность в контрольных точках спектра.

## Используемые технологии

- **Python**
- **MEEP**
- **NumPy**
- **Matplotlib**
- встроенный материал **`fused_quartz`** из `meep.materials`

## Физическая модель

Рассматривается нормальное падение плоской волны из воздуха на поверхность плавленого кварца.  
Коэффициент отражения сравнивается с аналитическим выражением:

R=({n_1-n_2}/{n_1+n_2})^2

где:
- (n_1 — показатель преломления воздуха;
- (n_2 — показатель преломления кварца, зависящий от длины волны.

В MEEP расчёт отражения выполняется по схеме из **двух прогонов**: сначала пустая ячейка, затем расчёт с материалом и вычитанием падающего поля. Такой подход прямо описан в туториале MEEP для air–silica interface.
